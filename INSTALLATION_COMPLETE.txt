A: 1. Go to Firebase Console â†’ Firestore â†’ Rules
   2. Set to test mode (allow all access)
   3. Make sure Anonymous auth is enabled
   4. Try clearing localStorage and refresh

   Just start adding data and they'll appear!

Q: "Data not syncing"
A: 1. Check Firebase Console for documents
   2. Verify collection path is correct
   3. Clear localStorage (Dev Tools â†’ Application)
   4. Try refreshing the page

   2. Click on Firestore Database
   3. Click "Data" tab
   4. Look for "artifacts" collection
